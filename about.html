About the blog

I feel that tutorials are great, but building projects is the best way to learn. I could have created this blog using wordpress or jekyll. Instead, I built the blog from scratch as a learning project. I can now learn from tutorials and apply it to the blog. The blog project can be a journal of the learnings as well. Also, nothing beats the joy of making things work with code!

In my quest to learn, I use tools and tech that this simple blog site does not need. Hence, this site is definitely, going to be "over-engineered".

The blogs by ponyfoo and jlongster were inspirations behind this.
