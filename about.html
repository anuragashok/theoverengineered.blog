About

The author of the blog is Anurag Ashok.

I am a software developer primarily focused on Java-based microservices. I am currently also exploring the javascript ecosystem for web development. I am lazy in repetitive tasks and choose automation. I love all things code, and am very passionate about "everything as code"

I live by the these quotes from famous software craftsmen :

Any fool can write code that a computer can understand. Good programmers write code that humans can understand.

― Martin Fowler

It is not enough for code to work.

Truth can only be found in one place: the code.

― Robert C. Martin

I feel that tutorials are great, but building projects is the best way to learn. I could have created this blog using wordpress or jekyll. Instead, I built the blog from scratch and made it a learning project. I can now learn from tutorials and apply it to the blog. The blog project can be a journal of the learnings as well. Also, nothing beats the joy of making things work with code!

In my quest to learn, I will use tools and tech that this simple blog site may not even need. Hence, this site is definitely, going to be "over-engineered".

I love feedback. Let me know if you have any. Thanks!
